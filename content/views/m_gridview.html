M.GridView

A layout component for arranging views in a grid.
Basics · Screenshots · Properties · Methods · Example · CSS Styles

Basics

The M.GridView is a component, that allows you to display several views horizontally aligned. Therefore you can either use a predefined layout or you can provide a custom layout.

Properties
Name Relevance Type Default value Description
layout mandatory Object null The layout for the grid view. There are two predefined layouts available:
- M.TWO_COLUMNS
   a two column layout, width: 50% / 50%
- M.THREE_COLUMNS
   a three column layout, width: 33% / 33% / 33%
- M.FOUR_COLUMNS
   a four column layout, width: 25% / 25% / 25%

To specify your own layout, you will have to implement some css classes and then define your layout like:

cssClass: 'cssClassForWholeGrid',
columns: {
    0: 'cssClassForColumn1',
    1: 'cssClassForColumn2',
    2: 'cssClassForColumn3',
    3: 'cssClassForColumn4',
    //........
}

Example

Setting up a default two column grid. 

M.GridView.design({
    childViews: 'label1 label2',
    layout: M.TWO_COLUMNS,
    label1: M.LabelView.design({
        value: 'Column 1'
    }),
    label2: M.LabelView.design({
        value: 'Column 2'
    })
})

Setting up a default four column grid. 

M.GridView.design({
    childViews: 'label1 button label2 image',
    layout: M.FOUR_COLUMNS,
    label1: M.LabelView.design({
        value: 'Column 1'
    }),
    button: M.ButtonView.design({
        value: 'Column 2'
    }),
    label2: M.LabelView.design({
        value: 'Column 2'
    }),
    image: M.ImageView.design({
        value: 'theme/images/column4.png'
    })
})

Setting up a custom three column grid. 

M.GridView.design({
    childViews: 'label1 label2 label3',
    layout: {
        cssClass: 'container',
        columns: {
            0: 'column1',
            1: 'column2',
            2: 'column3'
        }
    },
    label1: M.LabelView.design({
        value: 'Red'
    }),
    label2: M.LabelView.design({
        value: 'Green'
    }),
    label3: M.LabelView.design({
        value: 'Blue'
    })
})

When you are using a custom layout, is shown in the code sample above, you will need to add the css classes defined with the layout property as follows. 

div.container div {
    text-shadow:none;
    color: #FFFFFF;
    font-weight:bold;
}
div.container .column1, div.container .column2, div.container .column3 {
    float: left;
    width:25%;
    text-align:center;
}
div.container .column1 {
    background-color: #FF0000;
    width:50%;
}
div.container .column2 {
    background-color: #00FF00;
}
div.container .column3 {
    background-color: #0000FF;
}

CSS Styles

When you are using the grid view with one of the default layouts, the following CSS classes (applied on div elements) are relevant for you:

  • ui-grid-a: This defines the outer box of the two column grid.
  • ui-grid-b: This defines the outer box of the three column grid.
  • ui-grid-c: This defines the outer box of the four column grid.
  • ui-block-a: This defines the first column of the grid.
  • ui-block-b: This defines the second column of the grid.
  • ui-block-c: This defines the third column of the grid.
  • ui-block-d: This defines the fourth column of the grid.

M.ToggleSwitchView

A toggle switch view. You can think of it as an boolean switcher.
Basics · Screenshots · Properties · Methods · Events · Example · CSS Styles

Basics

A binary "flip" switch is a common UI element on mobile devices that is used for binary on/off or true/false data input. You can either drag the flip handle like a slider or tap one side of the switch.

Example

As described, there are several properties available for configuring a ToggleSwitch. The following examples show how to use them properly.

  1. A example with all available properties. 
    childViews: 'toggleswitch1 toggleswitch2 toggleswitch3',

toggleswitch1:M.ToggleSwitchView.design({
    label: 'I am the label and this switch has isMini:YES',
    isMini: YES
}),

toggleswitch2:M.ToggleSwitchView.design({
    label: 'I am the label and this switch has isMini:NO',
    isMini: NO
}),

toggleswitch3:M.ToggleSwitchView.design({
    label: 'Select language',
    isMini: NO,
    onLabel: 'DE',
    offLabel: 'EN',
    offValue: 'en',
    onValue: 'de',
    isInline: YES,
    cssClass:'lang',
    events:{
        change:{
            action: function(){
                console.log('change');
            }
        }
    }
}),

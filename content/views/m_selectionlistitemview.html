M.SelectionListItemView

Add selectable items to a selection list.

Properties
Name Relevance Type Default value Description
isSelected optional Boolean This property states whether a selection list item is selected or not.
label optional String This property can be used to specify a label for a selection list item. If set, the label will be displayed instead of the value, so you can use the item's value as an internal value.
E.g. if you use a selection list to select a color, you could set an item's value to '#FF0000' but its label to 'Red'. If there is no label specified, the value is displayed instead.
value mandatory String This property can be used internally to identify an item. If no label is defined for the item, the value is also used as label.

Example

The items are defined inside the definition of a M.SelectionListView. 

M.SelectionListItemView.design({
    value: 'ff0000',
    label: 'red',
    isSelected: YES
});

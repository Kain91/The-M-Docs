M.ListItemView

The template for every list entry as a container for different other views.
Basics · Screenshots · Properties

Basics

This is the prototype for any list item view. It can only be used as child view of a list view (M.ListView).

Properties
Name Relevance Type Default value Description
type String M.ListItemView Returns the type. Use only as getter!
hasSingleAction optional Boolean YES This property determines whether a list item has one single action that is triggered once there is a click anywhere inside the list item or if there are specific actions defined for single ui elements within one list item.
deleteButton optional M.ButtonView M.ButtonView.design({ icon: 'delete',value: '' }) This property contains the list item's delete button that is automatically shown if the list view's built-in toggleRemove() functionality is used.
isDivider optional Boolean NO This property determines whether the list item is a divider or not.
recommendedEvents recommended Array ['tap'] This property specifies the recommended events for this type of view.
isSelectable optional YES YES This property can be used to specify whether a selection list item can be selected or not. Note, that this only affects styling stuff. If set to NO, you still can apply e.g. tap events.

M.CanvasView

A plain canvas element with some helper methods.
Basics · Screenshots · Properties · Methods · Example · CSS Styles

Basics

The canvas view basically renders a simple canvas tag into the DOM. Additionally it offers some wrappers for canvas-based methods, but mostly you will just use this view for the first rendering of the canvas element and then work on the dom element itself.

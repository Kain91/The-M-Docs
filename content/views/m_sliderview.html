M.SliderView

A touch-optimized slider view.
Basics · Screenshots · Properties · Methods · Example · CSS Styles

Basics

The slider view renders a touch-optimized slider that can be used to set a number within a specified range.

Example

Setting up a canvas view. 

canvas: M.CanvasView.design({

})

Setting the width and height of a canvas within a controller. 

var canvas = M.ViewManager.getView('myPage', 'canvas');
canvas.setSize(400, 300);

Working on a canvas with plain JavaScript. 

var canvas = M.ViewManager.getView('myPage', 'canvas').getCanvas();
var context = M.ViewManager.getView('myPage', 'canvas').getContext();

context.fillStyle = "red";
context.fillRect(25, 25, 100, 100);

M.DashboardItemView

The template for every list entry as a container for different other views.
Basics · Screenshots · Properties · Methods · Events · Example · CSS Styles

Basics

A dashboard itm view contains an icon and a label and can be used as the only kind of child views for a dashboard view. While a dashboard view specifies the container of a dashboard, the dashboard item view is used to specify the actual visible items on that dashboard. They can be attached to a dashboard as either static child views or dynamically via content binding.

Properties
Name Relevance Type Default value Description
icon recommended String The path/url to the dashboard item's icon.
label optional String The label for the dashboard item. If no label is specified, the value will be displayed instead.
value mandatory String This property specifies the dashboard item's value. If there is no label set, this value is used as a visual text.

Example

Setting up dashboard items via content binding: 

// define the dashboard view and configure the content binding
M.DashboardView.design({
    contentBinding: {
        target: MyApp.MyController,
        property: 'items'
    }
})

// set the content within a controller (e.g. triggered by a pageshow event)
var items = [{
    icon: 'theme/images/icon_settings.png',
    label: 'Settings',
    value: 'settings'
},
{
    icon: 'theme/images/icon_edit.png',
    label: 'Edit',
    value: 'edit'
},
{
    icon: 'theme/images/icon_wifi.png',
    label: 'WiFi',
    value: 'wifi'
}];
this.set('items', items);

Use tap events with dashboard items: 

var items = [{
    icon: 'theme/images/icon_settings.png',
    label: 'Settings',
    value: 'settings'
},
{
    icon: 'theme/images/icon_edit.png',
    label: 'Edit',
    value: 'edit',
    events: {
        tap: {
            target: this,
            action: function(id) {
                var item = M.ViewManager.getViewById(id);
                var value = item.value;
                var label = item.label;
                // now do something with it...
            }
        }
    }
},
{
    icon: 'theme/images/icon_wifi.png',
    label: 'WiFi',
    value: 'wifi'
}]

CSS Styles

There are a few CSS classes that define the layout and the look and feel of a list item view:

  • tmp-dashboard-item: This class is responsible for the arrangement of the items on the dashboard.
  • tmp-dashboard-item-label: This class defines the layout and styling a list item's label.

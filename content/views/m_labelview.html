M.LabelView

The basic view for placing text in your UI. (read-only).
Basics · Screenshots · Properties · Methods · Events · Example · CSS Styles

Basics

The M.LabelView is basically a view to display a text. E.g. the text that is often displayed in a M.ToolBarView is a LabelView inside this bar. The value of the label can be statical or dynamical. It's also a common task to localize the value of a label. It can also used as a link inside the application and to external URLs.

Properties
Name Relevance Type Default value Description
cssClass optional String This property can be used to set CSS classes for a label.
hyperlinkTarget optional String This property can be used to specify a hyperlink target for this label. It only works in combination with the hyperlinkType property.
hyperlinkType optional String This property can be used to specify a certain hyperlink type for this label. It only works in combination with the hyperlinkTarget property. Three types are available:
  • M.HYPERLINK_EMAIL, for linking to an e-mail adress.
  • M.HYPERLINK_WEBSITE, for linking to a URL.
  • M.HYPERLINK_PHONE, for linking to a phone number.
isInline optional Boolean NO Can be used to let a label float with another following label. Both labels need to have it set to YES.
newLineToBreak optional Boolean YES Determines whether a new line '\n' within the label's value should be transformed into a line break '
' before it is rendered.
tabToSpaces optional Boolean YES Determines whether a tabulator '\t' within the label's value should be transformed into four spaces ' ' before it is rendered.
value mandatory (if computedValue is not used) String This property defines the value of the label, which means the text that is displayed.

Example

 

M.LabelView.design({
    value: 'Hello, World',
    cssClass: 'bigLabel'
});

A label with a content binding: 

M.LabelView.design({
    value: '',
    cssClass: 'bigLabel',
    contentBinding: {
        target: myApp.myController,
        property: 'labelText'
    }
});

/* activating content binding in a controller action */
...
this.set('labelText', 'Hello, World');

A label using computed value: 

M.LabelView.design({
    cssClass: 'dateLabel',
    computedValue: {
        contentBinding: {
            target: myApp.myController,
            property: 'timestamp'
        },
        value: '',
        operation: function(v) { /* format timestamp to a readable date string */
            return M.Date.create(v).format('dd.mm.yyyy HH:MM:ss');
        }
    }
});

/* activating content binding in a controller action */
...
this.set('timestamp', 1307720438815);

A label representing a hyperlink to a website: 

M.LabelView.design({
    cssClass: 'urlLabel',
    value: 'Go to www.the-m-project.org',
    hyperlinkTarget: 'http://www.the-m-project.org',
    hyperlinkType: M.HYPERLINK_WEBSITE
});

No CSS classes are added to the LabelView by default. Style it by passing classes to it via the cssClass property.

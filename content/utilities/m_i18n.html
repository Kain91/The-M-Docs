Basics

With M.I18N, The-M-Project provides a tool for internationalisation and localisation. It can be used to set and get the application's language and to localize any string within an application. Such a string, for example, can be a label, a url or a path to an image.

To get an idea of what is possible with this feature, take a look at our Todos demo application and switch the language.

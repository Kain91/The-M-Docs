Basics

The M.Math object wraps JavaScript's Math object and extends it with several own methods. It offers methods for:

  • expontentiate
  • retrieving random numbers in a range
  • rounding

↑ top

Methods
Name Parameters Return Type Description
pow
  • base: The base.
  • exponent: The exponent.
Number: The result of the exponentiation operation. This method returns the value of the base to the power of the exponent. So e.g. pow(2, 3) would return '2 to the power of 3' --> 8.
random
  • min: The minimal value.
  • max: The maximal value.
  • Number: A random number between the passed range.
The method returns a random number within the range given by the min property and the max property, including the min and max value.

A test with 100.000 iterations for random(1, 3) created the following distribution:
- 1: 33.2%
- 2: 33.2%
- 3: 33.6%
round
  • input (as Number): The input value.
  • type (as String/Constant): The type of rounding. Needs to be one of these three:
    • M.FLOOR: Returns the next lower integer, so 2.1 and 2.9 both would return 2.
    • M.CEIL: Returns the next higher integer, so 2.1 and 2.9 both would return 3.
    • M.ROUND: Returns the nearest integer, so 2.1 would return 2 and 2.9 would return 3.
  • decimals (as Number, optional): The number of decimals (only available for M.ROUND).
  • Number: The rounded number.
The method returns rounded version of the given input number. The type of rounding is defined by the second parameter: type. With the optional third parameter decimals, you can specify the number of decimal digits to be returned. For example round(1.2345, M.FLOOR, 3) would return 1.234. The default for this parameter is 0.

Examples

Raising a number to higher power 

M.Math.pow(2,3)
8

Retrieving random numbers in a range

Getting a random between 1 and 10: 

M.Math.random(1,10);
4

Rounding a number

Rounding a number to the nearest integer with rounding type M.ROUND. 

M.Math.round(32.1, M.ROUND);
32

Rounding a number to the next higher integer with rounding type M.CEIL. 

M.Math.round(32.1, M.CEIL);
33

Rounding a number to the next lower integer with rounding type M.FLOOR. 

M.Math.round(32.82, M.FLOOR);
32

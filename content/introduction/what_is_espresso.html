Basics

Espresso can be seen as The-M-Project's little friend. Basically you would call it a build tool, but it is more. Espresso is aware of both The-M-Project and your application. It provides tools for initializing an application and for creating certain components.

One of the most important features of Espresso is the built-in server. It is Node-based and, once started, builds your application on-the-fly whenever you request a specific url with any browser inside your local network. This allows you to test an application on any relevant mobile device without having to deploy it to a remote server.

Some more facts about Espresso:

  • Espresso is a Node-based build tool.
  • Espresso is The-M-Project's little friend.
  • Espresso is a server for the development process.
  • Espresso is packing up application.
  • Espresso is minimizing the amount of data of your application.

What Espresso is not!

Especially when it comes to Node, there are a lot of rumours and misunderstanding concerning Espresso. So here are a few thing, that Espresso is not:

  • Espresso is no requirement for running The-M-Project apps.
  • Node is no requirement for running The-M-Project apps.
  • Espresso is no application server for productual use.

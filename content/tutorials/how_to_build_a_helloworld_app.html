Setup a new project

First of all, make sure you downloaded Espresso&The-M-Project and did all the initial stuff described in the Get Started section on our website.

Now lets create a new project with Espresso. Therefore type the following into your command line ... 

espresso help

... to see all possible commands: 

Available commands:
build          Command to build a project
deploy         Command deploy projects
generate       Command to generate new or additional files for a project
init           Command line tool to generate a new project
package        Command to build native applications
server         Command to run the Development server
version        Command to print the Espresso version
help           Command to show help for a subcommand

The init command looks promising, don't you think? So lets see what it can do. 

espresso init help

We've got the following options: 

Available options:
-p, --project=PROJECT       Project name
-d, --directory=DIRECTORY   Specify a custom projects directory Default: $PWD
-e, --example               Generate an example "Hello World" application Default: false
-h, --help                  Show this help for command init Default: false

So we will need the -p option for specifying a project name and the -e option for setting up a hello world example. Let's do it. So first of all navigate to the directory where you want to init your project in, then run the just learned command: 

cd /path/to/folder
espresso init -e -p HelloWorldSample

Espresso now will initialize your project, copy all the required resources and set up some sample code, since we chose the -e option.

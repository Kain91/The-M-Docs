Requirements

You need to have git and nodeJS installed on your machine.

Installing git: http://help.github.com/git-installation-redirect/
Installing nodeJS: http://nodejs.org/#download (for windows users)

↑ top

Make a directory

Make your own The-M-Project director named themproject and go there 

mkdir themproject
cd themproject
↑ top

Get Espresso

Get Espresso from GitHub or download the prepacked version.
Place Espresso inside of the 'themproject' folder you just created.
The prepacked version additionally contains all sample apps of The-M-Project. So, you don't have to checkout anything from our git repository. To start right away, go on to step 4.)

Getting Espresso from github: 

git clone git://github.com/mwaylabs/Espresso.git 
cd Espresso/ 
git submodule update --init

Getting Espresso via npm:

npm install espresso -g

Create an alias to have the espresso command available (insert the path to Espresso on your machine! We recommend to use ~/.bash_profile file):

vim ~/.bash_profile
alias espresso='/path/to/Espresso/bin/espresso.js'
↑ top

Get the sample apps

Get The-M-Project Sample Apps from GitHub.
Make sure, that the folder with the sample apps is placed in the same directory as Espresso.
The 'themproject' folder should contain a Espresso folder and the sample apps folder.


Getting the sample apps from github:

git clone git://github.com/mwaylabs/The-M-Project-Sample-Apps.git
↑ top

Run the app

To run a sample app - for example the KitchenSink App - browse to the KitchenSink subfolder in the sample apps directory ... 

cd The-M-Project-Sample-Apps/KitchenSink/

... and run: 

espresso server

This will call Espresso to build the app and start the built-in web server. 

Server running at http://127.0.0.1:8000/KitchenSink
↑ top

Handle the app

Open your web browser (Google Chrome or Safari are recommended) with the URL above and enjoy the KitchenSink App.
